Open Source Altitude Adjustment Knob


spacer

This fixed knob is a replacement for the altitude adjustment handle on SkyWatcher EQ6-R Pro and Atlas II mounts. It provides a more positive engagement, making altitude adjustment easier. It replaces in seconds, and will not strip like the stock handle does. 3D print your own or order here.

Developer: Steven Bellavia

Comes with:
  • 1x Carbon-Filled PETG 3D Printed Knob. This is a strong, durable material that should last a lifetime.
  • 1x M5x20 Stainless Steel Socket Head Cap Screw

    • Installation:
    1. Remove the existing handle using a 4mm hex (Allen) wrench.
    2. Gently tap the knob on using a block of wood and a hammer. The fit is intentionally tight.
    3. Using the supplied M5X20 socket head screw, gently tighten with the 4mm hex wrench.
